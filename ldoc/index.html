Functions

parse (file)
Parse INI file and return table structure

Parameters:

  • file (string) Path to INI file

Returns:

    (table|nil) Parsed table or `nil` if file not found

Usage:

    local cfg = ini.parse("config.ini")
print(cfg.server.host, cfg.server.port)
save (tbl, path)
Save a Lua table into an INI file Handles sections, arrays, and global variables

Parameters:

  • tbl (table) Data to write
  • path (string) Destination file path

Returns:

    (boolean) `true` on success, `false` on error

Usage:

    ini.save({
  server = { host = "127.0.0.1", port = 8080 },
  log = { level = "debug" }
}, "config.ini")
